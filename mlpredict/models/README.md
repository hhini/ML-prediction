# 模型文件说明

## 目录用途

本目录用于存放健康风险预测模型文件。应用会自动检测并加载本目录中的模型文件。

## 支持的模型格式

应用支持以下格式的模型文件：

- `.pkl` - Pickle格式
- `.pickle` - Pickle格式
- `.joblib` - Joblib格式
- `.model` - 通用模型格式

## 模型要求

1. **输入特征顺序**：模型必须接受以下特征顺序的输入：
   
   1. `如果使用马桶，是否习惯盖马桶盖` (0=否, 1=是, None=缺失)
   2. `家庭厕所类型` (0=传统旱厕, 1=冲洗坑厕/测, 2=抽水马桶)
   3. `居住房屋所有权` (0=否, 1=租房, 2=自建房, 3=二手房, 4=新房)
   4. `零食的食用频率` (0=否, 1=1-2次/年, 2=1-2次/月, 3=1-2次/周, 4=3-5次/周, 5=＞5次/周)
   5. `家中蔬菜的购买方式` (0=自家种植, 1=超市, 2=菜市场, 3=街头小贩/都有)

2. **预测方法**：
   - 分类模型：需实现 `predict_proba()` 方法，返回概率数组
   - 回归模型：需实现 `predict()` 方法，返回预测值

3. **模型序列化**：模型必须可以通过以下方式加载：
   ```python
   # Joblib格式
   import joblib
   model = joblib.load('model.joblib')
   
   # Pickle格式
   import pickle
   with open('model.pkl', 'rb') as f:
       model = pickle.load(f)
   ```

## 放置方法

1. 将训练好的模型文件复制到本目录
2. 确保模型文件扩展名符合要求
3. 启动应用后，系统会自动检测并加载模型

## 示例

```
models/
├── model.pkl          # Pickle格式模型
└── README.md          # 本说明文件
```

## 注意事项

- 请确保模型文件大小合理，避免过大影响加载速度
- 模型文件必须与训练时的特征处理逻辑一致
- 如果模型加载失败，应用会在控制台输出错误信息
- 建议使用scikit-learn、XGBoost、LightGBM等常见机器学习库训练模型
